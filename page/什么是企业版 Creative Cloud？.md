# 什么是企业版 Creative Cloud？

企业版 Creative Cloud 为您的企业提供了设计卓越客户体验所需的一切工具，适用于任何设备或触点。通过连接桌面和移动应用程序以及云服务，您可以高效地创建和协作。此外，高级加密和安全部署选项可以保护您的内容。

## 主要功能

- **跨桌面和移动设备创作**：赋能团队随时随地创建，快速灵活地设计工作流程。企业版 Creative Cloud 包括 Adobe Photoshop CC 和 Illustrator CC 等桌面应用程序，以及让您在移动中开始项目并在桌面上完成项目的连接移动应用程序。
- **跨应用程序和团队共享资产**：Adobe CreativeSync 是 Creative Cloud 的核心技术，它将您的文件、字体、图像、视频、设置和其他资产放在桌面、网络和移动应用程序中的指尖。
- **获得所需的安全性和控制**：通过易于使用的 Adobe Enterprise Dashboard 管理部署和许可证。通过静态和传输加密保护您的创意资产，或将它们存储在防火墙后面。使用联合 ID/单点登录进一步保护您的知识产权。
- **利用企业支持**：获得部署和许可证管理的入职协助和 24x7 的专门技术支持。此外，无限次的专家服务培训课程帮助创意团队充分利用 Creative Cloud。

## 部署和管理变得简单

企业版 Creative Cloud 通过灵活、安全的部署选项和高级加密，为 IT 部门提供了简化的解决方案。
